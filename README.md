# Asset Physical Status – D365 F&O Customization

## 📌 Overview
This project demonstrates a custom development in Microsoft Dynamics 365 Finance & Operations.

The goal was to create a configurable system for managing asset physical statuses used in maintenance and RMA processes.

---

## ⚙️ Features
- Custom table for Asset Physical Status
- Use of Extended Data Types (EDTs)
- Unique index to enforce business rules
- Form for managing statuses (CRUD operations)
- Integration into Asset Management module

---

## 🏗️ Technical Highlights
- Developed using Visual Studio + D365 F&O
- Data modeling using EDTs (best practice)
- Form design using SimpleList pattern
- Menu item integration
